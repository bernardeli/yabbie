Yabbie

Yabbie is a gem to wrap SlimerJS.

Source Download
Tweet

Installation

Add this line to your application's Gemfile:

gem 'yabbie', github: 'net-engine/yabbie'

And then execute:

$ bundle

Or install it yourself as:

$ gem install yabbie

Dependency

It relies on SlimerJS, and a script that contains what you want to achieve. For the script, you can check our default.js. It does nothing but takes a png snapshot. It is just so that Yabbie does not fail if you don't provide one. Please don't rely on it.

Usage

Yabbie::Slimer.new(html_file, options).to_file(file)

For more usage, please investigate in the code while we don't build a nice README
